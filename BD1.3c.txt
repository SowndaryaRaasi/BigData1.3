Linux Commands
  pwd:  gives the present working directory
  vi: The default editor that comes with the UNIX operating system is called vi (visual editor).
  mkdir: to create a directory
  rm: used to remove a file from directory
  ls: it is used to list the files.
 echo: Used to display text on screen.
 cat: allows us to create single or multiple files, view contain of file, concatenate files and redirect output in terminal or files.
 mv: moves a file
 cp: copies a file
 date: shows the current date and time. 
 cal: shows a calendar of the current month. Use e.g., 'cal 10 1995' to get that for October 95, or 'cal 1995' to get the whole year. 
 cd:change the directory.
